## Clone the repository
git clone -b main https://github.com/vedanshpaliwal/CryptoListing.git

### `npm start`
Install the dependencies

### `cd client/npm start`
start react app

### `cd server/npm start`
start the server

## Tools, Languages & Libraries used:
* ReactJS
* MongoDB
* NodeJs/ExpressJs
* CSS
* NOMICS API
